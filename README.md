# 🛡️ ThreatGuard - Malicious URL Detection

A Flask web application that detects malicious URLs and files using Machine Learning and Google Gemini AI.

## 🚀 Features
- 🔗 URL Classification (Benign / Phishing / Malware / Defacement)
- 📄 File Content Analysis (PDF / TXT)
- 🔐 Secure User Authentication
- 📊 Scan History Logging

## 🛠️ Tech Stack
- Python & Flask
- Google Gemini 2.5 AI
- SQLite Database
- HTML, CSS, JavaScript

## ⚙️ How to Run
1. Clone the repository
   git clone https://github.com/anuragjaiswar950-byte/ThreatGuard-Malicious-URL-Detection.git
2. Install dependencies
   pip install -r requirements.txt
3. Run the app
   python MAIN0.PY
